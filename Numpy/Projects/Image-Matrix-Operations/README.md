# 🖼️ Mini Project 01: Image Matrix Operations using NumPy

## 📌 Project Overview

This project demonstrates how digital images can be represented and manipulated using NumPy arrays. It focuses on applying NumPy concepts to perform basic image processing operations without using advanced computer vision libraries.

---

## 🎯 Objectives

- Understand image representation as NumPy arrays.
- Apply NumPy array operations to images.
- Practice indexing, slicing, broadcasting, and array manipulation.
- Build a strong foundation for Computer Vision and Deep Learning.

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pillow (PIL)
- Matplotlib

---

## ✨ Features Implemented

- 📷 Load an image
- 🔢 Convert image into a NumPy array
- 📐 Display image properties
- 🔄 Horizontal Flip
- 🔄 Vertical Flip
- 🔁 Rotate Image (90°)
- ✂️ Crop Image
- ☀️ Increase Brightness
- 🌑 Decrease Brightness
- ⚫ Convert to Grayscale
- 🎭 Generate Negative Image
